# College2Career

🚀 A comprehensive Training and Placement Management System built using Django, Bootstrap, and SQLite to bridge the gap between students, placement officers, and recruiters.

## 📖 Overview

College2Career is a web-based platform designed to streamline the placement process in educational institutions. It provides a centralized system for managing student profiles, company recruitment drives, job applications, placement records, and resume management.

The platform helps students discover opportunities, enables recruiters to find suitable candidates, and assists placement officers in efficiently managing the entire placement workflow.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/College2Career.git
```

2. Navigate to the project directory

```bash
cd College2Career
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Apply migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

5. Run the development server

```bash
python manage.py runserver
```

6. Open in browser

```text
http://127.0.0.1:8000/
```
